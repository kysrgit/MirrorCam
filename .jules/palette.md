## 2024-03-21 - Better Touch Feedback and Accessibility for Composite Custom Buttons
**Learning:** In Flutter, wrapping a `Container` with `GestureDetector` deprives users of native touch ripple feedback and screen reader context. When dealing with composite elements (like an icon in a circle and text below it), replacing `GestureDetector` with `Semantics(button: true)` + `Material(color: Colors.transparent)` + `InkWell` (with an appropriate `borderRadius`) provides a much better UX. Additionally, replacing the inner visual `Container` with an `Ink` widget allows the ripple to properly splash *over* the child's background color.
**Action:** Always prefer `Material` + `InkWell` + `Ink` (with `Semantics`) over `GestureDetector` + `Container` for custom buttons to ensure consistent visual feedback and accessibility.
