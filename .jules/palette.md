## 2024-05-18 - Improve Button UX and Accessibility with Material + InkWell + Ink
**Learning:** Using `GestureDetector` with a `Container` for custom buttons omits native touch ripple feedback and accessibility properties. Using `Semantics(button: true)` + `Material(color: Colors.transparent)` + `InkWell` + `Ink` instead maintains native interactions, adds keyboard focus styling, ensures proper ripple bounding constraints, and improves screen reader accessibility. When applying an `InkWell` ripple effect over a composite widget, use an appropriate bounding `borderRadius` (e.g., `BorderRadius.circular(16)`) to prevent the ripple from misaligning with the rectangular layout bounds.
**Action:** When creating custom composite buttons in Flutter, always prefer the `Semantics` + `Material` + `InkWell` + `Ink` pattern over `GestureDetector` to ensure standard UX and accessibility guidelines are met.
